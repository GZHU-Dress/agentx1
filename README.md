# [壹代理](http://bitbucket.org/CrazyBoyFeng/agentx1) #

针对星网锐捷私有 802.1X 网络认证协议进行双向代理。

## 简介 ##

项目工作在 Linux 核心的目标平台，致力于改善认证的兼容性和稳定性，以适配多种版本的协议为首要开发目标。

原理是在网络认证过程中以代理的身份转发服务器与客户端传输的数据时进行微调，并在认证成功后尝试中继，不用关心具体认证数据和加密算法。
